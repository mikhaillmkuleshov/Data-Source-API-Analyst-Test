# Content Folder

This folder contains documentation and output samples for the GitHub API test.

## Files
- `api_documentation.md`  
  Endpoints used, request structure, parameters, pagination, rate limits, and response notes.

- `troubleshooting_guide.md`  
  Practical guide for common API problems (401/403/429/5xx) and how to fix them.

## Samples
All JSON samples are stored in: `Samples/`
- `search_repositories_sample.json`
- `commits_sample_page1.json`
- `commits_sample_paginated.json`
- `repo_contents_root_sample.json`
